This is a simple web-based Warehouse Management System (WMS) designed for small businesses like pizzerias. It allows you to:

Register products with name, barcode, and quantity

Scan barcodes using your webcam

View and filter your current stock list

Delete products from stock

Save data locally (browser LocalStorage) so it persists after page reloads

(Optional) Export the stock list to CSV (feature coming soon)

Technologies Used
HTML

CSS

JavaScript

QuaggaJS (For live barcode scanning)

How to Use
Open index.html in your browser (preferably Google Chrome or Firefox).

Go to the "Product Registration" tab to add new products.

Use your webcam to scan barcodes, or enter them manually.

View your current stock in the "Stock" tab.

Filter products by name or barcode.

Delete products if needed.

Your stock list will be saved automatically in your browser.
